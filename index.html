O que é um compilador ?

  • Tradutor: linguagem de entrada para linguagem destino
  • Em geral é dividido em camadas
    • Análise Léxica
    • Análise Sintática
    • Analise Semântica *
    • Geração de Código
* Normalmente onde ocorre a geração de código intermediario

O que é um compilador

Entradas e Saidas

  • Classicamente a entrada é um arquivo escrito em uma linguagem de programação e a saida é um executável
  • Mas podem existir outras aplicações:
    • Análise de cadeias Quimicas Lazar
    • Correção de Estilo Uncrustify
    • Transpilação Babel
    • Antivírus MalwareBytes
